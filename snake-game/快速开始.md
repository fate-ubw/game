## 🌐 部署到 GitHub Pages

### 第一步：推送代码

```bash
# 在 game 目录下
cd /Users/jim/0-Phd/Back-to-future/Web2Tool/webNavigator-homepage/game

# 添加所有文件
git add .

# 提交
git commit -m "feat: 添加贪吃蛇游戏"

# 推送到远程仓库
git push origin main
```

### 第二步：启用 GitHub Pages

1. 打开你的 GitHub 仓库页面
2. 点击 **Settings**（设置）
3. 在左侧菜单找到 **Pages**
4. 在 **Source** 下选择 **GitHub Actions**
5. 保存后，GitHub 会自动构建和部署

### 第三步：访问你的游戏

部署完成后访问：`https://YOUR_USERNAME.github.io/game/`

## ⚙️ 游戏设置

- **难度选择**: 简单、中等、困难
- **暂停**: 空格键或点击暂停按钮
- **重新开始**: 游戏结束后点击重新开始按钮

## 🔧 自定义配置

### 修改仓库路径

如果你的 GitHub 仓库名不是 `game`，需要修改 `vite.config.js`：

```javascript
export default defineConfig({
  base: '/your-actual-repo-name/',  // 改成你的仓库名
  // ...
})
```

### 修改游戏参数

在 `src/components/SnakeGame.vue` 中可以修改：
- `canvasWidth` / `canvasHeight`: 画布大小
- `gridSize`: 网格大小
- `speedMap`: 不同难度的速度
